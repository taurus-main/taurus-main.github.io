1. Executive Summary

Taurus DEX is an AI-driven, intent-based, cross-chain aggregated decentralized trading platform designed to create the next generation trading experience: users simply express their goals, such as "I want to buy ETH with 100 USDT," and the system recognizes the intent, aggregates routes, and automatically executes optimal trades.

Traditional DEXs suffer from complex operations, fragmented chains, and uncontrollable slippage. Taurus solves different trading needs for both Web3 newcomers and experts through semantic input + aggregated routing + CopyTrade.

AI Intent Recognition

Natural language understanding of user trading intentions

Cross-Chain Aggregation

One-stop multi-chain asset trading experience

Smart Copy Trading

Replicate professional trader strategies

Our goal is to become the "one-stop intelligent trading brain" for all wallets, DApps, and community portals.

3. System Architecture & Design

Taurus DEX adopts a modular architecture design, ensuring each core function can be deployed independently while collaborating efficiently. The overall structure is as follows:

Intent Recognition Layer (Intent Engine)

Parses user input (such as "buy ETH with 100U") and converts it into structured trading objectives.

Routing Aggregation Layer (Routing Aggregator)

Coordinates multiple DEX aggregators (1inch, Odos, Jupiter, LiFi) to filter optimal routes.

Solver Network (Solver Layer)

Selects on-chain executors (Solvers) to complete final trade calls, ensuring security and verifiability.

Strategy Recommendation Layer (Strategy Layer)

Provides trading suggestions and strategy combinations based on user profiles and intents.

Status Feedback Module (UX Layer)

Returns execution status, routes, cost estimates, and results, supporting on-chain queries.

Security Guarantee: All operations are completed through non-custodial structure, with user assets always remaining in their own wallets. The system is compatible with mainstream wallets (such as MetaMask, TokenPocket, OKX Wallet, etc.) and cross-chain environments (EVM + Solana + zkSync).

4. AI Strategy Module

The AI Strategy Engine is one of the core modules of Taurus DEX, dedicated to helping users make reasonable token selection and fund allocation even when unfamiliar with market conditions.

Core algorithms include:

Hotness Score (HScore)

Builds real-time hotspot scoring based on 24-hour DEX volume growth, Twitter discussion volume, and token holder address growth.

Risk Rating (RScore)

Analyzes liquidity depth, price volatility, and token lock status to distinguish between high and low-risk targets.

Policy Recommendation Model (PSM)

Combines user intent to recommend combinations like "Stable Portfolio (ETH + BTC)" or "High Potential Portfolio (Meme + ETH)".

Example Scenario

User Input: "I want to swap 100 USDT, just don't lose money."
System Recommended Portfolio:
  • 70 USDT → ETH (Mainstream Stable)
  • 30 USDT → $WHY (On-chain Hotness + High Potential)

Tech Stack: All models are based on LLM fine-tuning + RLHF human feedback mechanisms, with future plans to introduce local inference deployment and user-autonomous model fine-tuning.

5. CopyTrade Module

The Taurus CopyTrade system is open to ordinary users, strategy providers (KOLs), and institutional addresses, implementing decentralized "copy trading" functionality through on-chain verifiable trading behaviors.

Functional Structure:

Follow Address

Copy real-time trading behaviors of any address (such as Smart Money) with proportional control execution (10%~100%).

Follow Person (KOL)

Platform-verified on-chain strategists who can display returns, follower count, historical trades, and open following permissions.

Follow Strategy

"Strategy combinations" published by Taurus or third parties, such as "Weekend Low Volatility Portfolio" or "Hot Token 3-Day Rotation Strategy".

Incentive Mechanism:

  • Strategy publishers receive up to 15% revenue sharing, settled proportionally by follower execution trades
  • System displays monthly/weekly/ROI rankings to increase credibility and exposure
  • Supports creating "exclusive strategy groups" where fans join following via referral codes

The CopyTrade module supports EVM multi-chain deployment and will open API access for Bot users and institutional deployment in the future.

7. Business Model & Revenue Sources

Taurus DEX's business model is built around three cores: user growth, strategy ecosystem development, and executor incentives.

Revenue Sources:

Aggregated Trading Rebates

Taurus aggregates trades from protocols like 1inch, Jupiter, LiFi and receives official rebate income

CopyTrade Management Fees

KOL copy trading behaviors generate up to 15% revenue sharing, with platform taking partial service fees

8. Risk Control & Compliance Mechanisms

Taurus DEX emphasizes Web3 security compliance standards, designing security mechanisms across frontend, contract, and API layers:

Frontend Restrictions

Default restrictions on access from high judicial risk regions like USA, North Korea

Contract Logic Audits

Platform conducts third-party audits of all core contracts with regular reviews to avoid potential vulnerabilities

Executor Whitelist System

All Solvers must register and pass verification to access execution modules

MEV Protection

Default access to Flashbots RPC, MEV Blocker channels to prevent front-running and slippage losses

Data Privacy Protection

Intent content signature submission, receipt encryption processing to protect user on-chain intent privacy

Taurus is committed to long-term trustworthy operation, ensuring all trade executions are transparent on-chain and user-autonomous.

10. Community Growth & Incentive Systems

Taurus community incentive system is designed around "Contribution Equals Equity", aiming to drive protocol growth and ecosystem building through user participation.

Points System (Bull Points)

  • Swap operations, copy trading behaviors, strategy publishing, user invitations, check-in tasks all earn "Bull Points"
  • Bull Points determine airdrop priority, advanced feature unlocks, and future staking rights weights

Referral System

  • Each user can generate exclusive referral codes to invite other users to register, use Taurus products and bind
  • Referrers receive partial Bull Points rebates and incentive revenue sharing in CopyTrade earnings

Leaderboards & Competition Mechanisms

  • System regularly publishes "Monthly Strongest Strategy", "Highest Return Copy Trader", "Most User Invitations" leaderboards
  • Top performers receive additional Bull Points, airdrop rewards, or token reward packages

Community Co-building Mechanism

  • Support establishing "Taurus DAO sub-communities", each can create subdomains and bind multiple KOLs
  • DAOs can apply for operational budgets or governance proposal permissions to jointly participate in platform building
